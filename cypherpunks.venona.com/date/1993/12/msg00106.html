A new public keyserver is now available.  

Send mail with "help" in the subject line to:

	public-key-server@martigny.ai.mit.edu

The server support mail access only, not FTP access.

Questions concerning this keyserver should be addressed to:

	public-key-server-request@martigny.ai.mit.edu

Requests to automatically synchronize this keyserver with other
keyservers should be sent to the above address.
