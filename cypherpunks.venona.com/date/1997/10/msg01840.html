Security Policies To Go!

Don't re-invent the wheel. Get the most comprehensive source of information
security policies available. Information Security Policies Made Easy
(version 6) is a collection of 840 already-written policies in book and
CD-ROM. ISPME is the fastest and most expedient way to write new and /or
revise existing information security policies. Use your favorite word
processing package and cut and paste from these ready-to-go policies,
justification statements, and other supporting materials. Generate a
complete policy statement in five minutes or less.

It Includes extensive instructions which provide practical advice about the
policy development process. Applicable to organizations throughout the
world, the book/CD-ROM provides policies covering a broad range of topics
including: Internet, intranets, firewalls, network security, contingency
planning, computer viruses, electronic payment systems, web surfing
activity monitoring, Java, macroviruses, data classification, push
information distribution, software development, intellectual property
rights, password management, and much more. The latest version also
includes ten skeleton policies which are ready-to-go complete policy
statements. These skeleton policies can be used as is or expand them with
other policies from the book. ISPME provides everything needed to write
standards and communicate them in a clear, concise manner.
