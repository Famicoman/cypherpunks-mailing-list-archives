II.  Tip o' the Day

Faster 'Seeker

If you have a fast connection, such as a fractional T1 or faster, you 
can dramatically decrease the amount of time a search takes by 
increasing the number of simultaneous connections.  We suggest 
increasing the number of simultaneous connections to 25 or 35.  
Experiment with more...
(You can always decrease the number of connections)

Steps:
1.  Click Tools/Preferences
2.  Select the Network tab (or in Version 3.0, Search Limits)
3.  Change the number of Simultaneous Connections
4.  Click OK
